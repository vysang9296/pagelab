# 📁 Public Binder (퍼블릭 바인더)

**Public Binder**는 대용량 로컬 파일 및 문서 페이지를 비파괴적으로 제어, 편집, 검색, 정렬할 수 있는 강력한 데스크톱 파일 관리 플랫폼입니다. 본 프로그램은 크게 페이지 단위 비파괴적 편집 도구인 **Page Lab**과 고성능 로컬 파일 분류/심층 검색 엔진인 **Folder Lab**의 두 가지 독립적인 작업 공간을 제공합니다.

서버 통신 없이 100% 사용자의 PC 로컬 환경에서 모든 작업이 처리되어 기업의 망분리 환경이나 극비 보안 문서 취급에 최적화되어 있습니다.

---

## 🌟 주요 모듈 및 핵심 기능 (Modules & Features)

### 1. 📄 Page Lab (페이지 랩)
여러 형식의 문서들을 불러와 원본 훼손 없이 페이지를 가상으로 해체하고, 분류/회전/병합하여 최종 배출하는 도구입니다.

* **다양한 포맷 자동 변환**: PDF, HWP, HWPX 등의 다기능 파일들을 드래그 앤 드롭으로 업로드 시 백엔드에서 사용자 PC의 한컴오피스 엔진을 사용해 자동으로 PDF 변환 및 썸네일을 렌더링합니다.
* **비파괴적 페이지 제어**: 썸네일을 마우스로 잡고 분류 폴더로 Copy-on-Drag 하여 배정합니다. 페이지 90도 회전, 임시 제외(Dim)를 지원하며 원본은 절대 손상되지 않습니다.
* **표준 다중 선택 지원**: Shift 키(연속 범위 선택), Ctrl 키(개별 추가/제외) 조합을 완벽히 지원하여 수백 장 규모의 문서를 신속하게 분류할 수 있습니다.
* **상세한 내보내기 옵션**: 분류된 폴더 단위로 통합 PDF 저장, 낱장 ZIP 압축 저장, 혹은 다중 선택된 폴더들을 계층 그대로 이중 ZIP 파일(ZIP 속 ZIP)로 패키징할 수 있습니다.
* **개별 페이지 네이밍 (Context Menu)**: 내보내기 전 우클릭으로 각 페이지의 최종 출력 파일명을 미리 지정할 수 있어 다운로드 후 파일명을 다시 바꾸는 번거로움이 없습니다.

---

### 2. 🗂️ Folder Lab (폴더 랩)
대규모 로컬 저장소와 가상 작업 공간을 연동하여, 문서를 원하는 구조로 재배치하고 본문 심층 검색을 통해 지식 자산을 체계화하는 3단 레이아웃 탐색기 공간입니다.

* **3단 분할 레이아웃 (Total Commander Style)**:
  * **좌측 패널**: 로컬 드라이브의 파일 시스템을 트리 구조로 신속하게 탐색(지연 로딩 적용).
  * **중앙 전송바**: 좌측에서 선택한 대량의 파일/폴더들을 우측 가상 스테이징으로 한 번에 전송.
  * **우측 패널 (듀얼 모드)**:
    * **가상 스테이징 작업**: 가상의 중첩 계층 트리 구조(Nested Tree) 내에서 폴더 구조를 임의로 조작(새 폴더 생성, 인라인 이름 수정, 드래그 앤 드롭 노드 재배치).
    * **실시간 로컬 작업**: 실제 로컬 드라이브를 연결하여 파일 복사, 이동, 삭제(안전한 Windows 휴지통 SHFileOperationW 연동) 등의 작업을 직접 실시간 반영.
  * **하단 드로어**: 심층 본문 검색 기능 및 텍스트 미리보기 제공.
* **SQLite Trigram FTS5 기반 심층 문서 검색**:
  * 한글 텍스트 검색에 특화된 Trigram 토크나이저를 채택하여 문서 본문 내 검색어를 고속으로 탐색합니다.
  * `.hwp`, `.hwpx`, `.pdf`, `.pptx`, `.xlsx`, `.docx`, `.txt`, `.md` 등 비정형 포맷들의 본문 텍스트 자동 파싱 및 인덱싱.
  * 검색 결과 매칭 구문 하이라이팅 스니펫(Snippet) 및 클릭 시 본문 텍스트 즉시 미리보기 제공.
* **체크박스 선택 부분 내보내기**:
  * 가상 스테이징 영역에서 원하는 폴더 및 파일만 체크한 뒤 `ZIP 내보내기` 또는 `로컬 동기화`를 수행하면, 선택한 항목들만 그 계층 구조를 보존한 채 부분 배출됩니다.
  * 로컬 동기화(Commit) 시 지정 폴더 하위에 `Staging_Export` 폴더를 자동 생성하며, 기존 폴더가 존재할 경우 순번을 자동 증가(`Staging_Export(1)`)하여 기존 자료를 안전하게 보호합니다.
* **편리한 탐색기 UX**:
  * 우클릭 컨텍스트 메뉴를 통한 실제 파일 실행 및 탐색기 열기 연동.
  * 우클릭 컨텍스트 메뉴로 새 폴더 생성 시, 전체 트리가 닫히지 않고 변경된 부모 경로 하위 자식 노드만 부분 갱신(Partial Refresh)되어 트리 확장 상태(하이어러키)가 완벽하게 유지됩니다.

## 🛠️ 기술 스택 (Tech Stack)

* **Backend**: Python 3.10+, PyMuPDF (fitz), SQLite3 (FTS5 + Trigram), ctypes (Windows Shell API), win32com.client (한컴오피스 COM Automation), Pywebview 4.0+
* **Frontend**: Vanilla JavaScript, HTML5 (Drag & Drop API), CSS3 (Google AI Studio Inspired Theme), SortableJS
* **Packaging**: PyInstaller (Windows Standalone Executable Spec 구성)

## 💻 설치 및 구동 방법

### 1. 사전 요구사항
* OS: Windows 10 / 11 (64bit)
* Python 3.10 이상
* **한글(한컴오피스) 필수 설치** (HWP/HWPX 문서 PDF 변환 엔진에 필수적입니다.)

### 2. 개발 및 테스트 실행
```bash
# 1. 의존성 패키지 설치
pip install -r requirements.txt

# 2. 로컬 디렉토리 실행
python main.py
```

### 3. 단일 실행 파일 (.exe) 빌드
```bash
# PyInstaller 패키징 진행
pyinstaller PageLab.spec
```
빌드가 완료되면 `dist/PageLab` 폴더 내에 윈도우 실행 본이 생성됩니다.
